# LMS-BY-AK

A Learning Management System (LMS) project developed using PHP (XAMPP) and MySQL.

## 🚀 Features

- User authentication (Login/Register)
- Student and teacher dashboards
- Course management system
- Upload and manage learning materials
- Assignment handling
- Admin control panel
- File upload support (with Git LFS for large files)

## 🛠️ Tech Stack

- PHP
- MySQL
- HTML, CSS, JavaScript
- Bootstrap
- XAMPP (Local Server)
- Git LFS (for large files)
